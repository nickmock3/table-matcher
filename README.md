# table-matcher

MVP bootstrap for:
- Next.js (App Router)
- Cloudflare Workers (OpenNext)
- D1 + R2
- Better Auth (Google OAuth)
- Drizzle ORM

## Quick start
```bash
bun install
bun run db:generate
bun run dev
```

Cloudflare bindings + OAuth 実検証は:
```bash
bun run dev:cf
```

詳細手順: `docs/setup.md`
