# 空席マッチングwebサイト
## 基本コンセプト
飲食店の“空席”という無駄になりがちなリソースを、即時性 × 割安価格でマッチングする集客支援サイト。
### サービスの概要
本サービスは、現在の空席情報をリアルタイムで可視化し、来店を促すことで  
- 飲食店の空席ロス削減  
- 利用者の「今から行きたい」に応える  

ことを目的としたマッチング型集客サイトです。  

### 利用者側  
対象：個人の一般客  
利用シーン  
- 当日・直前予約  
- 数時間後の来店  
- 急に空いた時間での店探し  

特徴  
- 近隣エリアの「今空いている店」がすぐ分かる  
- 登録・手数料は不要  

### 飲食店側  
対象：個人経営の飲食店  
機能  
- 空席情報を手動で登録  

メリット  
- 空席を埋めるための即効性のある集客  
- 手数料無料  
- 後述するアフィリエイト収益の分配による追加収入  

  
※ 本サービス自体は予約機能を持たず、集客導線に特化するシンプルな構成

## MVPの範囲
やること
- 店舗一覧（トップ兼用）/検索（市区町村 / ジャンル / 空席のみ）
- 店舗詳細ページ
- 空席ステータス表示（空いている/空いていない）
- 地図表示
- 外部予約サイト誘導
管理
- 管理者ログイン（Google認証、管理者1名）
- 店舗登録・編集（ログイン用メール登録）
- 空席情報の登録・更新（店舗ユーザー/管理者）
- ログ収集（閲覧1imp + 滞在時間送信）
- 簡易Bot除外
- 日次・月次集計（JST）
- 管理者ダッシュボード（日別/月別）
- 管理者設定（曜日別分配比率 / 有効滞在時間 / 単価）
- 月次CSV出力（店舗別合計）
やらないこと
- サイト内での予約管理
- 自社での決済処理
- レビュー・評価
- リアルタイム精算
- 金融API・自動振込
- 複雑なAI分析
